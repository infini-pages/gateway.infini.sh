              response_body_truncate #

              描述 #

              response_body_truncate 过滤器用来截断过长的响应体。

              配置示例 #

              一个简单的示例如下：

              flow:
  - name: test
    filter:
      - name: response_body_truncate
        parameters:
          max_size: 1024

              参数说明 #

              名称类型说明
              max_sizeint最大的请求体消息长度，超过则进行截断处理，默认 1024
              add_headerbool是否添加一个标识消息已经被截断的 Header 信息 RESPONSE_BODY_TRUNCATED=%SOURCE_SIZE%->%TARGET_SIZE%，默认 true
